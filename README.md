# 🎓 EduGenius AI – Təhsildə İnqilab

EduGenius AI, həm şagirdlər, həm də müəllimlər üçün nəzərdə tutulmuş süni intellekt əsaslı öyrənmə ekosistemidir. Bu layihə təhsil prosesini avtomatlaşdırmaq və fərdiləşdirmək üçün yaradılmışdır.

## 🚀 Super Gücləri
* **Müəllim Asistenti:** Saniyələr içində dərs planı və imtahan suallarının hazırlanması.
* **Şagird Dostu:** Mürəkkəb məsələlərin addım-addım vizual və məntiqi izahı.
* **Fərdi Öyrənmə:** Hər bir şagirdin zəif tərəflərinə uyğun xüsusi tapşırıqlar.

## 🛠 Texnologiyalar
* **Dil:** Python 3.10+
* **Beyin:** Google Gemini Pro API / OpenAI GPT-4
* **Platforma:** GitHub / Streamlit (Veb interfeys üçün)

## 📁 Layihə Strukturu
- `/src`: Kodların ana qovluğu
- `/data`: Tədris materialları və bazası
- `/tests`: Sistemin dəqiqliyini yoxlayan testlər
